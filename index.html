Centrar Imagen en GitHub Pages

Diferentes métodos para centrar tu imagen "Untitled118_20250825195001-1.png" en una página web alojada en GitHub Pages.

Método Flexbox (Recomendado)

.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

Método CSS Grid

.contenedor {
  display: grid;
  place-items: center;
  height: 100vh;
}

Método Clásico

.contenedor {
  text-align: center;
}

.contenedor img {
  display: block;
  margin: 0 auto;
}
